Extending JS with tools

JavaScript is awesome, but even with EMCAScript 6, it lacks many, easy-to-implement but super useful functions.
JSPlus aims to exend JS in a quiet, yet powerful way.
It provides partial replacement to the outdated and slow JQuery libraries, ex. $( css );
It adds shuffeling, mixing arrays and numbers, can generate and transpose colors and provides stats for strings. Additionally, it implements a PHP $_GET style URL reader and writer; making easier to use less and less PHP and improving compatiblity.
It provides a modern, yet cross compatible modular system. Parts of NXT.JS that have not been linked initially, can be used later, on-demand, making initial website load quicker, especially on slow netowks.

Dependencies

Packages, fonts and misc. used by NXT JS for its design

Fonts

Several fonts are used by NXT.JS to ensure an appealing look.

  • Montserrat used for titles, headers and important, short texts.
  • Open Sans used for all kind of text. all styles imported.
  • Material Icons Font used for fonts, applied using i.material-icons. Not included by default, but advised to do so.

    • Design

    NXT.JS follows the material design principles outlined here.
    However, it is not a 100% implementation of its concepts, as material design has mainly been developed for mobile apps and later modified to be web compatible as well.
    If you want a one-on-one material design, use JSPlus with Material Light.
